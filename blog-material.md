# 전체 학기 시간표 생성기 - 블로그 소재 정리

## 한 줄 요약

일주일치 시간표를 입력하면 한 학기 전체 진도표 양식(날짜+요일+교시+커스텀 컬럼)을 자동 생성해주는 웹앱.

## 이 도구가 필요한 상황

- 학기 초, 진도표(수업 일지) 양식을 만들어야 할 때
- 매주 반복되는 시간표를 학기 전체 날짜에 펼쳐야 할 때
- 엑셀이나 노션에 붙여넣을 수 있는 표 형태가 필요할 때
- 수작업으로 날짜를 하나하나 입력하기 싫을 때

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 학기 기간 설정 | 시작일~종료일을 date picker로 입력 |
| 주간 시간표 입력 | 요일(월~일), 교시(1~7), 커스텀 필드를 행 단위로 추가 |
| 커스텀 컬럼 | 기본 컬럼(영역, 학급) 외에 컬럼 추가/삭제/이름변경 가능 |
| 자동 생성 | 입력한 주간 패턴을 학기 전체 날짜에 매칭하여 전체 행 생성 |
| CSV 다운로드 | BOM 포함 UTF-8 CSV 파일로 다운로드 (엑셀 한글 깨짐 방지) |
| 클립보드 복사 | TSV 형식으로 전체/선택 복사 (엑셀·노션 붙여넣기 가능) |
| 분할 복사 | 100행 단위로 나누어 복사 (노션 등 대용량 붙여넣기 실패 방지) |
| 행 선택 | 체크박스로 개별/전체 선택 후 선택 행만 복사·다운로드 |

## 사용 흐름

1. 학기 시작일과 종료일 입력
2. 출력 컬럼 설정 (기본: 영역, 학급 / 필요에 따라 추가·삭제·이름변경)
3. 주간 시간표 항목 추가 (요일, 교시, 각 컬럼 값 입력)
4. "시간표 생성" 클릭
5. 결과 테이블 확인
6. 전체/선택 복사 또는 CSV 다운로드

## 기술적 특징

- **100% 클라이언트 사이드**: 서버 API 없음. 모든 로직이 브라우저에서 실행됨. 백엔드·DB 불필요.
- **날짜 처리**: `YYYY-MM-DDT00:00:00` 형식으로 파싱하여 UTC/로컬 시간대 혼동 방지.
- **CSV 인코딩**: UTF-8 BOM(`0xEF 0xBB 0xBF`) 포함으로 엑셀에서 한글 깨짐 방지. `TextEncoder`로 명시적 UTF-8 인코딩.
- **엑셀 날짜 오인 방지**: `1-7` 같은 숫자-하이픈 패턴에 `'` 접두사를 붙여 엑셀이 날짜로 변환하는 것을 차단. 실제 날짜(`YYYY-MM-DD`)는 예외 처리.
- **분할 복사**: 100행 단위 청크로 나누어 순차 복사. 노션 등에서 대용량 붙여넣기 실패하는 문제 우회.
- **버전 관리 시스템**: `lib/changelog.ts`에 버전·날짜·변경사항을 구조화. `changelogs/` 폴더에 버전별 .md 파일 보관. Footer에서 모달로 업데이트 내역 표시.

## 디자인

- **디자인 시스템**: accent 컬러 `#D2886F`, 배경 white→subtle 그라디언트
- **폰트**: Pretendard Variable (CDN)
- **CSS 변수**: `--accent`, `--bg-white`, `--bg-subtle`, `--text-main`, `--text-sub`, `--text-light`, `--border-color`
- **다크모드**: `prefers-color-scheme: dark` 미디어 쿼리로 CSS 변수 전환 (자동 감지)
- **모바일 대응**: 반응형 레이아웃, 터치 타겟 최소 44px, 자동 zoom 방지(font-size 16px 강제), 가로 스크롤 차단
- **UI 전체 한글화**: 버튼·라벨·안내문·에러 메시지 모두 한글

## 제한사항 / 참고

- 공휴일·재량휴업일 자동 제외 기능 없음. 생성 후 행 선택·삭제로 수동 처리 필요.
- 시간표 데이터는 브라우저 메모리에만 존재. 새로고침 시 입력 내용 사라짐 (저장 기능 없음).
- 교시는 1~7교시까지 고정. 0교시나 8교시 이상은 선택 불가.
- 요일은 월~일 7일 모두 지원.
- 현재 버전: v1.1.1 (2026-03-06 기준).

## 대상 사용자

한국 초·중·고 교사. 학기 초에 진도표·수업 일지 양식을 만들어야 하는 교사.

## 만든 이유 (동기)

매 학기 시작할 때마다 교사들이 진도표 양식을 수작업으로 만든다. 주간 시간표는 정해져 있는데, 이걸 학기 전체 날짜에 하나하나 펼치는 작업은 단순 반복이다. 이 반복 작업을 자동화하기 위해 만들었다.
